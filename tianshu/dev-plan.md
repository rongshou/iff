## 一、项目目标 (已确认)

**核心定位:**
天枢是一个 **Streamlit Web 工具**,基于已收到的方法论框架,帮助学生/家长/顾问完成「测评 → 专业选择 → 生涯规划」全流程。

| 维度 | 决策 |
|------|------|
| 工具形态 | **Web 网页 (Streamlit)** |
| 目标用户 | **家长 / 学生 / 顾问三者皆可** |
| 输入来源 | **全部手工录入** |
| 输出格式 | **网页展示 + PDF 下载** |
| 数据持久化 | **保存用户档案(支持历史查看)** |
| 部署模式 | **多用户 + 后端数据库(SQLite)** |

> 💡 已收到的关联资料:`学生综合特质测评与专业选择、生涯规划全流程框架.pdf`(16 页)
> 这是天枢的**方法论核心**,所有功能模块都围绕这份框架的 7 大板块展开。
